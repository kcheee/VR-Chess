# 📃 프로젝트 소개

### Oculus Quest2를 사용하고 HandTracking을 사용해 Chess를 직접 집어 기물을 이동시킬수 있는 VR-Chess 게임 입니다.

**Game** : VR AI Chess   
**Game mode**: Human Player vs Computer   
**Difficulty** : high, medium, low   
**AI-Algorithms** : Alpha-beta pruning   
**Software** : Unity-3D   
**Language** : C#
